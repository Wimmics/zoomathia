## Migration toward TEI P5

en version P5 de TEI (actuellement dans data ils sont à la version P4 ou antérieure). Le fichier `p4top5.xsl` est une stylesheet qui permet de passer à la version 5 d'un fichier xml. Elle s'appuie notamment sur le logiciel "Saxon".
Pour appliquer la stylesheet dans Vs Code, il faut faire un Ctrl + alt + T dans l'éditeur sur un texte xml. Il faut avoir préalablement installer une extension ainsi qu'un logiciel et avoir configuré l'extension pour qu'elle pointe vers la stylesheet et vers le logiciel qui applique cette stylesheet.
